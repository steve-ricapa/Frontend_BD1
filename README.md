# Interbank Frontend Project

## Database Connection
Este proyecto está configurado para conectarse a una base de datos PostgreSQL local.
Los detalles de conexión se encuentran en `src/lib/db.js` y `.env`.

**Credenciales:**
- Host: `localhost`
- Puerto: `5433`
- Usuario: `postgres`
- Base de datos: `proyecto_bd`

**Nota:** La conexión real se realiza a través de un backend intermediario (`server.js`) o herramientas de desarrollo, ya que el navegador no puede conectar directamente a PostgreSQL.
